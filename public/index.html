Regional Time API

Get current time for any timezone around the world

Welcome to Regional Time API

This API provides current time information for various regions and timezones around the world. Use it to get accurate local time for any supported region.

Interactive API Documentation

API Endpoints

Get Current Time for a Region

GET /api/time/{region}

Returns the current time for a specific timezone region.

Example: /api/time/Africa/Casablanca

List Available Regions

GET /api/regions

Returns a list of all available timezone regions that can be used with the API.

Example: /api/regions

Example Usage

To get current time in New York:

GET /api/time/America/New_York

To get current time in London:

GET /api/time/Europe/London

To get current time in Tokyo:

GET /api/time/Asia/Tokyo
